Import Bulk Pricelist Using Excel or CSV File Odoo Apps

Do you want to import multiple price list for vendor, customer and product? Don't worry, here is your solution. This Odoo Apps helps to import Vendor pricelist, import Sale Pricelist, import product pricelist with multiple pricelist lines using CSV and XLS file by one-click with same configuration provided on odoo. After installing this Odoo apps only you have to manage your pricelist on xls and csv file then you can easily import them into Odoo ERP.

If you want to import extra fields from default odoo apart from our sample files, we will add those on our import module without any extra cost. You can purchase our module and contact us on sales@browseinfo.
e.,g If you want to add company on pricelist, we will add without any cost.
